# Webpack Boilerplate
My Webpack Boilerplate for large projects

# Development
To develop locally
1. git clone this repo
2. `cd webpack-boilerplate`
3. `npm install`
4. `npm run serve` to run the development server
5. Follow the instructions in your terminal or open `localhost:8080`
